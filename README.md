## Installing (Kali Linux):

```
apt-get update

git clone https://github.com/israfilkhanworld/anonymous.git

cd anonymous

ls

chmod +x anonymous.py

python anonymous.py

```


## Installing (Termux):

```
pkg update && pkg upgrade -y

pkg install git -y

pkg install python -y 

git clone https://github.com/israfilkhanworld/anonymous.git

ls

cd anonymous

chmod +x anonymous.py

python anonymous.py

```
